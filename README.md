# Dreamize LMS

> A high-performance, industry-aligned Learning Management System (LMS) designed to bridge the gap between education and industry through expert mentorship and structured learning programs.

---

## Overview

Dreamize is a Next.js-based LMS platform that connects students with expert trainers. The platform supports three user roles: **student**, **trainer**, and **admin**.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Tech Stack

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **Backend**: Express.js, MongoDB, Mongoose
- **Auth**: JWT-based authentication
